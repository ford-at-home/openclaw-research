# openclaw-research

**Agent commands:**
| Command | What it does |
|---------|-------------|

| `/research:add-research <topic>` | Agent-curated research with taxonomy |
| `/research:find-research <query>` | Search the library |
| `/research:update-index` | Rebuild INDEX.md manually |
